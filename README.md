## Description

A simple CRUD GraphQL API using [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
yarn
```

## Running the app

1. Start the docker-compose to set up the database:

```bash
docker-compose up
```

2. On a different terminal tab, start the application

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
yarn start:prod
```

## Test

```bash
# unit tests
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```
